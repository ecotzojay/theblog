# The Blog | Review Site
### ART 272 - Design For The Web
Note: Not a responsive design

## Project Prompt & Objective
The goal of this project was to build a unique multi-page blog from the ground up featuring reviews or recommendations on a topic of choosing. This assignment followed an industry-standard workflow, requiring the establishment of a target audience and content before building high-fidelity wireframes in prototyping software.

### Core Requirements
* Two-Page Website - Development of at least two separate HTML pages.
* 3 Review Entries - Each entry must consist of an original image and a word review or article.
* Originality - All content must be original assets, including all writing and images.
* Standard Elements - Implementation of a consistent navigation menu, footer, and a short introductory paragraph.

## My Implementation
I created a blog dedicated to souvenir shopping in Paris, specifically reviewing streets that offer the best options for college students on a budget.

* Content Focus - Reviews of various Parisian streets, comparing and contrasting them based on staff friendliness, prices, hours, store size, and variety.
* Visual Identity - The design experience was inspired by traditional Parisian street signs, utilizing a navy and dark green color palette to create a cohesive aesthetic.
